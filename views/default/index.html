Semantic Web Exporter

Introduction

This application exports machine-readable information about your facebook profile and friends, in an open format (RDF), described using the Friend of a Friend (FOAF) and Semantically-Interlinked Online Communities (SIOC) vocabularies.

Why would you want to do this? See The Future of Online Social Networking.

Instructions

  1. If you have a "foaf:Person" URI for yourself already, it can be reused. If not, don't worry, and leave the first field blank.
  2. Decide whether you want to export information about just yourself, or include friends as well.
  3. Select a preferred serialization format. RDF/XML is the most common, but others are easier to edit manually if you want to make changes.
  4. Use the download link to save the export file.
  5. Upload the saved file to your website.
  6. Provide a link on your blog to the file, like so (assuming RDF/XML serialization):
    • <link rel="meta" type="application/rdf+xml" title="FOAF" href="foaf.rdf" />

Note: The download button provided on this page works only once, for your privacy. Please reload the page if you wish to generate data with different options. All data is created dynamically, and is not saved on servers outside of Facebook.
