Semantic Web Exporter

Introduction

This application exports machine-readable information about your facebook profile and friends, in an open format (RDF), described using the Friend of a Friend (FOAF) vocabulary.

Instructions

  1. Use the tabs below to select a preferred serialization. RDF/XML is the most common, but others are easier to edit manually if you want to make changes.
  2. Use the download link to save the full export file. Only a couple of friends are included in the preview below, the download includes all friends.
  3. Upload the saved file to your website.
  4. Provide a link on your blog to the file, like so (assuming RDF/XML serialization):
    • <link rel="meta" type="application/rdf+xml" title="FOAF" href="foaf.rdf" />
